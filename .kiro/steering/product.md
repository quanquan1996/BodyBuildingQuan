# 产品概述

姿态对比 (Pose Comparison App) 是一个 Flutter Web 应用，用于对比参考图片与用户照片之间的身体姿态。

## 核心功能

- 上传参考图片（如健身动作、舞蹈姿势）
- 上传用户模仿该动作的照片
- 使用 MediaPipe Pose Landmarker 自动检测姿态
- 可视化叠加对比两个姿态：
  - 用户骨架（绿色实线）
  - 参考幽灵骨架（青色虚线）
  - 误差向量（红色连线）

## 目标用户

健身爱好者、舞者、运动员，或任何想要对比自己身体姿态与参考姿态的用户。

## 主要特性

- 基于浏览器运行（无需安装）
- 通过 MediaPipe 实时检测姿态
- 支持交互式缩放查看对比结果
- 响应式布局，适配桌面和移动端
